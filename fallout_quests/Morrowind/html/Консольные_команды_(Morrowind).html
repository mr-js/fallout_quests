Основные коды

Установка навыков и атрибутов

Целая серия кодов, работающих по одному принципу. Они модифицируют одну из характеристик игрока. Set устанавливает её напрямую, Mod добавляет значение, Get даёт посмотреть текущее значение. После команд Set и Mod, разумеется, вводится число, например: SetLevel 10.

Player->(1) (2), где (1)-одно из нижеуказанных значений, (2)-значение. Например Player->SetSpeechcraft 100
